名词

特征大写。会作为第一格(N)主语，第四格(A)宾语承受者。重要的是拼写、复数形式、性、意义。名词分为阳性(der)、中性(das)、阴性(die)。

冠词

1. 不定冠词 (一个，第一次提及)

阳性 中性 阴性
N ein ein eine
A einen ein eine

2. 定冠词 (在叙述前已经可以明确)

阳性 中性 阴性 复数
N der das die die
A den das die die

3. 零冠词

代词

1. 人称代词

N (第一格) A (第四格)
我 ich mich
你 du dich
他/它/她 er / es / sie ihn / es / sie
我们 wir uns
你们 ihr euch
他们 sie sie
您 Sie Sie

2. 物主代词 + 名词

原始形式
我: mein
你: dein
他/它/她: sein/sein/ihr
我们: unser
你们: euer
他们: ihr
您: Ihr

根据需要表示的名词的性和承担的成分对原始形式变形。

Ex:

阳性 中性 阴性 复数
N mein mein meine meine
A meinen mein meine meine

特殊: euer → eure, euren (去掉r前的e对于有e变形的变形)
